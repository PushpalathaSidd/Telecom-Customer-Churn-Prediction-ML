# Telecom Customer Churn Prediction using Machine Learning

## Project Overview
This project focuses on predicting customer churn using Machine Learning techniques on a telecom customer dataset. The workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation using Logistic Regression.

The main objective of this project is to identify customers who are likely to leave the telecom service, helping businesses improve customer retention strategies.

## Machine Learning Model
- Logistic Regression

The model was trained to classify whether a customer is likely to churn or not based on customer-related features.

---

## Model Performance
- Accuracy Score: *78.8%*

The model showed good performance in predicting customer churn using unseen test data.

---

## Key Insights
- Customers with month-to-month contracts showed a higher churn rate.
- Long-term contract customers had better retention rates.
- Data preprocessing and encoding improved model training performance.

---

## Project Conclusion
This project successfully demonstrated how Machine Learning can be used to predict telecom customer churn. Logistic Regression helped identify customer retention patterns and classify churn behavior effectively. The project also highlights the importance of preprocessing, feature selection, and model evaluation in building predictive analytics solutions.
